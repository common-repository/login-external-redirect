This plugin can redirect non users or not signed in users to any external or internal url.

== Description ==
During development phase you can use this plugin instead of showing \"Under Construction\" page. This plugin will redirect all visitors to a specified url and allow developers or admin to login via wp-admin panel.

Advantages of using this plugin from Under Construction Plugin
1> Your old site is still accessible when your new site is in construction state
2> Admin or developer can login using site-url/wp-login.php 
3> Easy to configure
4> Can even redirect to external urls

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/login-external-redirect` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Use the Settings->Login Redirect screen to configure the plugin
4. Check the Redirect checkbox and input url. 
5. Click Save button

DONE! Simple as like that.
